# 関連
- SVN

# 概要
コマンドラインで基本的な操作をします。

# 方法
カレントディレクトリ下の差分状況を確認する。
```
$ svn status
```

ここで状態が「?」の項目は追加する必要がある。
```
$ svn add (ファイル名)
```

カレントディレクトリ下の差分内容を確認する。
```
$ svn diff
```

カレントディレクトリ下の差分をコミットする。
```
$ svn commit --username (ユーザー名) -m "(メッセージ)"
```
