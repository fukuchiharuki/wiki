# 関連
- JSP
- JSTL
- Java

# 概要
JSP(EL式)で文字列を連結します。

# 方法
演算子「+」では文字列を連結できないので次のようにします。
```
<c:set var="hoge">${moge},${fuga}</c:set>
```

# 解説
タグに囲まれた箇所がvalue属性値の扱いになるようです。
