# キーワード
- awk
- 変数
- 置換

# したいこと

シェルスクリプトの中でawkを使い、シェルスクリプト中の変数をawkで利用したい。

# どうやって

{% raw %}
```
awk '{
        gsub("{USER_ID}", "'$USER_ID'");
        print $0;
}' $TEMPLATE
```
{% endraw %}

# ノート

gsubの$USER_IDのところ、

{% raw %}
```
gsub("{USER_ID}", "'
```
{% endraw %}

{% raw %}
```
$USER_ID
```
{% endraw %}

{% raw %}
```
'");
```
{% endraw %}

とシングルコートでawk分を分割する。

awk文の中で改行のエスケープは要らない。
