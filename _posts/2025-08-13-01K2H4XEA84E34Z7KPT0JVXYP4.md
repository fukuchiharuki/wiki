# キーワード
- Kotlin
- nullable
- Any
- cast

# したいこと

Any?を、warningを受けないように、リストにキャストするには一工夫要る。

# どうやって

Any?をList<Exception>?にする例

```
(value as? List<*>)
  ?.takeIf { list ->
    list.all { it is Exception }
  }
  ?.map { it as Exception }
```
