# 関連
- Excel

# 概要
Excelでランダムな文字列を作成します。新規ユーザーの作成で初期パスワードを設定しなければならないこと、ありますよね？そんなときのために。

# 方法

{% raw %}
```
=CHAR(INT(RAND()*26)+65)&CHAR(INT(RAND()*26)+97)&INT(RAND()*10)
```
{% endraw %}

# 解説
- 大文字

{% raw %}
```
CHAR(INT(RAND()*26)+65)
```
{% endraw %}

- 小文字

{% raw %}
```
CHAR(INT(RAND()*26)+97)
```
{% endraw %}

- 数字

{% raw %}
```
INT(RAND()*10)
```
{% endraw %}

- 連結

{% raw %}
```
&
```
{% endraw %}
