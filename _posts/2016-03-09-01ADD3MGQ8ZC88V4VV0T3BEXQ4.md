# キーワード
- css
- min-height
- リセット

# 現象

```
min-height: auto !important;
```

で min-height がリセットできない（ブラウザがある）。

# 原因

auto がそもそも正しくないのか、対応してくれないブラウザがある。

# 対策

モダンな書き方なら次が正しいそう。

```
min-height: initial !important;
```

あるいは、

```
min-height: 0 !important;
```

にする。

```
min-height: 100% !important;
```

でもなぜか期待どおりの動作をする。
