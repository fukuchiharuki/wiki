# キーワード
- Angular2
- ディレクトリ構成

# したいこと

ディレクトリ構成

```
quickstart/
  app/
  node_modules/
  index.html
  systemjs.config.js
```

となっているものを、

```
index.html
  quickstart/
    app/
    node_modules/
    systemjs.config.js
```

とできないものか。

# どうやって

systemjs.config.jsをbaseURLを追記すると動いた。

```
System.config({
  baseURL: '/quickstart',
```

# ノート

SystemJSはモジュールローダーで次のメリットがある。

- <script>を大量に書かない
- 順番も気にしない
