# キーワード
- flyway
- IntelliJ

# 現象

IntelliJでSpring Boot + flywayを動かしているがflywayが次のエラーになる。

```
Found more than one migration with version 1
```

# 原因

マイグレーションファイルを整理してversion 1は重複していないが、buildに吐かれるものが残っていて衝突したっぽい。

# 対策

cleanする。

gradleタスクでcleanした。
