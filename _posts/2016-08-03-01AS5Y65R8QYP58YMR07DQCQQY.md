# キーワード
- Eclipse
- git clone
- svn checkout

# したいこと

git clone や svn checkout でローカルに取得したディレクトリをEclipseのプロジェクトとして開きたい。Eclipseのプロジェクトファイルがリポジトリに含まれていればそれを開くだけだが、ソースコードだけしかない場合やり方が異なる。

# どうやって

1. 新規プロジェクト作成の手順で始めて、

1. 「Use default Location」のチェックを外して「Location」に取得したディレクトリを指定する。
