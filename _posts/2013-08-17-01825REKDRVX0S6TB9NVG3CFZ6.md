# 手順

## ポートフォワーディングを使う

### トンネルする
次のようにしてssh接続する。

{% raw %}
```
ssh <接続設定> -L3690:127.0.0.1:3690
```
{% endraw %}

### ローカルホストに接続する
svnクライアントで次のURLにアクセスする。

{% raw %}
```
svn://localhost/<sample>
```
{% endraw %}
