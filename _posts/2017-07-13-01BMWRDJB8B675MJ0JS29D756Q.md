# キーワード

- npm
- install
- HTTP_PROXY

# 現象

{% raw %}
```
npm install -g bower
```
{% endraw %}

などができない。読みにも行っていない感じ。

# 原因

プロキシ設定できてないから。

# 対策

.bash_profileにHTTP_PROXYを設定しておく。

{% raw %}
```
export HTTP_PROXY=http://プロキシサーバー:ポート番号
```
{% endraw %}

# 備考

EclipseとかAtomでも個別に設定してあげる必要があったかと思う。
