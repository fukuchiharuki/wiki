# 現象
リモートからMySQLに接続できない。エラーメッセージは次。
```
Can't get hostname for your address
```

# 原因
あなたのアドレスからホスト名を取得できません。

# 対策
サーバ側のhostsファイルに接続元のIPアドレスを登録しておくことで解決できます。

- hosts
```
接続元のIPアドレス 何か適当な名前
```

あるいは名前サーバを立てるか、逆引きさせない手段もあるようです。
