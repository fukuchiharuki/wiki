# 関連

- IE (Internet Explore)
- JavaScript

# 現象
window.open(url, window)がIEでエラーになる。

# 原因
パラメータwindowに使えない文字(「-」が少なくとも使えない)がある。

# 対策
パラメータwindowに使えない文字を使わない。
