# キーワード
- Lubuntu
- 日本語入力

# したいこと

Lubuntuで日本語入力したい。

Lubuntuをインストールしたままの状態だと、日本語表示できるものの日本語入力できない。

# どうやって

## 構成

日本語を入力するには、インプットメソッド（日本語入力システム）とインプットメソッドフレームワークが必要なようです。
このうち、インプットメソッドフレームワークであるFcitxは既にインストール済みである。
ここでは、インプットメソッド（日本語入力システム）であるMozcをインストールする。

:Mozc|インプットメソッド（日本語入力システム）
:Fcitx|インプットメソッドフレームワーク

## 手順

```
$ sudo apt update
$ sudo apt install -y fcitx-mozc
```

ここでいちどログアウトしてからログインし直す。

# ちなみに

MozcはGoogle日本語入力のオープンソース版のようです。
