# 関連
- SVN

# 概要
リビジョン番号を指定してコミットしたファイルの一覧を取得します。

# 方法
こんなシェルを書いて
```
#!/bin/sh
svn log -r ${1} -v svn://svnサーバ/リポジトリ/ > ${1}.log
```
こう実行します。
```
$ ./log.sh リビジョン番号
```

# 解説
svn logです。
