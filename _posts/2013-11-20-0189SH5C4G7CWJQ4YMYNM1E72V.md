# キーワード
- MySQL
- TRUNCATE
- AUTO_INCREMENT

# ポイント
TRUNCATE すると AUTO_INCREMENT は最初から数えます。また1から始まります。

*公式の記述から引用:*
```
テーブル ハンドラは最後に利用された AUTO_INCREMENT 値を記憶していませんが、また最初から数えます。
```

DELETE FROM とは違うね。
