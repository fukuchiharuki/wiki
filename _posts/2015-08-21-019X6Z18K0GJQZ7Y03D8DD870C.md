# キーワード

- sleep
- usleep
- expr

# したいこと

ミリ秒単位で待つ処理を入れたい。

# どうやって

{% raw %}
```
usleep $(($RANDOM * 1000))
```
{% endraw %}

# ノート

{% raw %}
```
usleep
```
{% endraw %}

は100万分の1秒単位の指定で待つ。

{% raw %}
```
$RANDOM
```
{% endraw %}

で0から32767までの整数を取得することができるが小さいので、

{% raw %}
```
$((計算式))
```
{% endraw %}

で計算する。これはexprするのと同じ。
