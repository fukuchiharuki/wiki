# キーワード
- wc
- test

# したいこと

- 行数をカウントして
- その行数がいくつであることをチェックする。
- チェックした結果は終了ステータスにする。

# どうやって

{% raw %}
```
cat なになに | grep なになに | wc -l | test $(cat -) -eq 0
```
{% endraw %}

# ノート

{% raw %}
```
wc -l
```
{% endraw %}

で行数をカウントすることができる。パイプを使えば標準出力をそのまま使用してカウントする。

{% raw %}
```
test
```
{% endraw %}

でチェックする。このコマンドは終了ステータスを返すのみ。

{% raw %}
```
$(cat -)
```
{% endraw %}

で標準入力を値として使用する。
