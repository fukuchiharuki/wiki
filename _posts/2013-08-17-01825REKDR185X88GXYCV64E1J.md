# 関連

- echo

# 概要
改行せずにecho出力する。

# 方法

{% raw %}
```
$ echo -n -e 'hoge'
```
{% endraw %}

# 解説
オプションの意味は以下の通り

- -n
  - 改行しない指定
- -e
  - 文字列指定
