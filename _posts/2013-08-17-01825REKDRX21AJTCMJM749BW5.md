# 概要
ブランチ（それぞれHEAD）間の差分を確認します（ファイル単位で）。

# 方法

{% raw %}
```
$ svn diff svn://svnサーバ/リポジトリ/ブランチい svn://svnサーバ/リポジトリ/ブランチろ --summarize
```
{% endraw %}

# 解説
summarizeがないとファイルの中も見ます。
