# 関連
- jQuery

# 現象
次のコードではtrオブジェクトを取得できない。

{% raw %}
```
$("table > tr")
```
{% endraw %}

マークアップで次のように書いていても。

{% raw %}
```
<table>
  <tr>
    <td>セル</td>
    <td>セル</td>
  </tr>
</table>
```
{% endraw %}

# 原因
DOM上、tableの子にtbodyを挿入するため。

マークアップで次のように書いたのと同じ。

{% raw %}
```
<table>
  <tbody>
    <tr>
      <td>セル</td>
      <td>セル</td>
    </tr>
  </tbody>
</table>
```
{% endraw %}

# 対策
tbodyを挟んで次のようにして取得する。

{% raw %}
```
$("table > tbody > tr")
```
{% endraw %}
