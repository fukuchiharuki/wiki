# 引数のデフォルト値
関数の引数についてデフォルト値を与えることはできないが次の手段で代替できる。

{% raw %}
```
function sample(arg1, arg2) {
}
```
{% endraw %}

を

{% raw %}
```
sample(1);
```
{% endraw %}

でコールした場合

{% raw %}
```
(typeof arg2 == 'undefined')?
```
{% endraw %}

で引数がなかったことが分かる。
