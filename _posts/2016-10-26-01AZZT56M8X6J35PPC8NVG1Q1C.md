# キーワード
- Angular2
- routerLinkActive

# したいこと

階層のあるメニューで、親の要素もハイライトしたい。

{% raw %}
```
├ メニュー☆
│ ├ サブメニュー☆
│ └ サブメニュー
```
{% endraw %}

# どうやって

単に、親にもrouterLinkActiveを書いてあげる。

{% raw %}
```
<li routerLinkActive="active">メニュー
  <ul>
    <li routerLinkActive="active"><a routerLink="/x">サブメニュー</a></li>
    <li routerLinkActive="active"><a routerLink="/y">サブメニュー</a></li>
  </ul>
</li>
```
{% endraw %}
