# 関連

- grep

# 概要
grepで空行やコメント行などを除外します。

# 方法

{% raw %}
```
$ cat filename | grep -v '#.*$' | grep -v '^$'
```
{% endraw %}

# 解説
オプション(-v)でマッチする行を除外した結果を出力します。
