# キーワード
- Titanium Mobile
- Alloy

# 目標
Window の初期化処理します。

# 方法
- Window.js
```
$.window.addEventListener('open', function(e) {
	// 初期化処理
});
```

# 解説
イベント open で処理します。イベント load だと子要素のロード前に動いてしまうのかダメです。
