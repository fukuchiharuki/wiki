# インストール

- Gradle ([https://gradle.org/](https://gradle.org/))
- Git for Windows ([https://git-for-windows.github.io/](https://git-for-windows.github.io/))
  - Git Bashを含めてインストールする
- Docker for Windows ([https://docs.docker.com/docker-for-windows/install/](https://docs.docker.com/docker-for-windows/install/))
  - Windows 7の場合はDocker Toolbox ([https://docs.docker.com/toolbox/overview/](https://docs.docker.com/toolbox/overview/))
- MySQL Workbench ([https://dev.mysql.com/downloads/workbench/](https://dev.mysql.com/downloads/workbench/))
  - サーバーはなし、クライアントだけでOK
- STS ([https://spring.io/tools/sts](https://spring.io/tools/sts))

# MySQL環境
Docker上にMySQLサーバーを立てる。ので、Git Bashで操作する。

## イメージをダウンロード

{% raw %}
```
$ docker pull mysql 
```
{% endraw %}

## コンテナを起動

{% raw %}
```
$ docker run \
--name hoge \
-e "TZ=Asia/Tokyo" \
-e MYSQL_ROOT_PASSWORD=rootpassword \
-e MYSQL_DATABASE=HOGE \
-e MYSQL_USER=hoge \
-e MYSQL_PASSWORD=hoge \
-p 3306:3306 \
```
{% endraw %}

## 停止と再起動

停止。

{% raw %}
```
$ docker stop hoge
```
{% endraw %}

再起動。

{% raw %}
```
$ docker start hoge
```
{% endraw %}

# プロジェクトを開く
## STS環境
### プラグイン

Help > Eclipse Marketplace...からインストールする。

- Buildship
- EGit
- Properteis Editor

### その他

- Lombok ([https://projectlombok.org/download.html](https://projectlombok.org/download.html))
  - ダウンロードしたjarを実行してインストールする

## ソースコードをクローンしてEclipse用に構成

{% raw %}
```
$ cd /path/to/project/
$ git clone https://***/hoge.git
$ cd hoge/
$ gradle eclipse
```
{% endraw %}

## STSで開く

1. 右クリック > Import...
1. Gradle/Existing Gradle Project
  - Next >
1. Project root directory: C:\path\to\project\hoge
  - Next >
1. Gradle distribution: Gradle wrapper
  - Next >
1. Finish

# Webアプリケーションを実行する
## STSから実行

1. Package Explorerでwebを選択状態にする
1. 虫マーク右の三角 > Debug As > Spring Boot App
  - Started WebApplication in *** secondsを確認
1. ブラウザから ttp://localhost:8080/ にアクセス

## Git Bashから実行

ビルド。

{% raw %}
```
$ gradle build
```
{% endraw %}

実行。

{% raw %}
```
$ java -jar web/build/libs/web-0.0.1-SNAPSHOT.jar
```
{% endraw %}

ブラウザから ttp://localhost:8080/ にアクセス。

### 文字化けしてるっぽいとき

環境変数を書き換える。

- /path/to/gradle/bin/gradle.bat

{% raw %}
```
set DEFAULT_JVM_OPTS="-Dfile.encoding=UTF-8"
```
{% endraw %}
