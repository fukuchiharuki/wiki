# 関連

- JavaScript

# 現象

{% raw %}
```
JavaScript: this.form.submit();
```
{% endraw %}

上記が走らない。

# 原因

{% raw %}
```
name="submit"
```
{% endraw %}

とかしてるのがいけない。（JavaScriptの予約語）

# 対策

{% raw %}
```
name="submit"
```
{% endraw %}

とかを消す。
