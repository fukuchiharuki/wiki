# 概要
タブ区切りのデータを出力します。

# 方法

{% raw %}
```
$ mysql -u(username) -p(password) (database) -B -N -e 'select * from (table);' > data.tsv
```
{% endraw %}

# 解説
各オプションは以下のとおり。

- -B
  - セパレータとしてタブを使う
- -N
  - カラム名を出力しない
- -e
  - クエリ指定
