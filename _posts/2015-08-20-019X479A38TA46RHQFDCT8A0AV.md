# キーワード

- sudo
- visudo

# フォーマット

{% raw %}
```
root    ALL=(ALL)       ALL
```
{% endraw %}

順番に、

1. 誰が
1. どこから（通常、ALL）
1. 誰の権限で
1. どのコマンドを実行することができるか

を設定する。

# 設定例

- ユーザーhogeが
- root権限で
- なんでもすることができる

{% raw %}
```
hoge    ALL=(root)      ALL
```
{% endraw %}

- グループmogeが
- root権限で
- シャットダウンすることができる

{% raw %}
```
%moge   ALL=(root)      /sbin/shutdown
```
{% endraw %}

# ノート

- グループを指定するには「%」を使う
- 複数のコマンドを指定するには「,」で連結する
