# 制御
- 新しいスクリーンを作成する

{% raw %}
```
C-tc
```
{% endraw %}

- スクリーンをトグルする

{% raw %}
```
C-tt
```
{% endraw %}

- スクリーンの一覧を表示する

{% raw %}
```
C-tw
```
{% endraw %}

- スクリーンを移動する

{% raw %}
```
C-t0 〜 Ct9
```
{% endraw %}

# 分割
- スクリーンを上下に分割する

{% raw %}
```
C-t S
```
{% endraw %}

- 分割したスクリーンのフォーカスを移動する

{% raw %}
```
C-ti
```
{% endraw %}

- 分割したスクリーンを閉じる

{% raw %}
```
C-t X
```
{% endraw %}

- 分割したスクリーンをフォーカスしているスクリーンだけ残す

{% raw %}
```
C-t Q
```
{% endraw %}
