# 制御
- 新しいスクリーンを作成する

```
C-tc
```

- スクリーンをトグルする

```
C-tt
```

- スクリーンの一覧を表示する

```
C-tw
```

- スクリーンを移動する

```
C-t0 〜 Ct9
```

# 分割
- スクリーンを上下に分割する

```
C-t S
```

- 分割したスクリーンのフォーカスを移動する

```
C-ti
```

- 分割したスクリーンを閉じる

```
C-t X
```

- 分割したスクリーンをフォーカスしているスクリーンだけ残す

```
C-t Q
```
