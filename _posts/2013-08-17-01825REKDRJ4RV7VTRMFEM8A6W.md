# 関連
- Tomcat
- XML

# 現象
XMLファイルについて次の警告が出る。

{% raw %}
```
No grammer constraints (DTD or XML schema) detected for the document.
```
{% endraw %}

# 原因
構造定義がないと言われている様子。

# 対策
次のようにルートの要素だけ書くとこの警告を回避できる。

{% raw %}
```
<!DOCTYPE hoge>
```
{% endraw %}
