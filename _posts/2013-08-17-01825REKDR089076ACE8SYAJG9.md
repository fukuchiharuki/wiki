# キャッシュにtmpディレクトリを使う
## 設定画面を開く
アドレス欄で次のように入力します。

```
about:config
```

すると「動作保証対象外になります！」と怒られます。ので「細心の注意を払って使用する」してください。
## キャッシュにつかうディレクトリを追加する
右クリック＞新規作成(W)＞文字列(S)、で次のキーを追加します。

```
browser.cache.disk.parent_directory
```

値を次のように設定します。

```
/tmp ※メモリをあてるように設定するとよい
```
