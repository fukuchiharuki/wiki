# キーワード
- WSL2
- Ubuntu
- アップグレード

# したいこと

WSL2のUbuntuが18系のため、主にnodeで不都合がある。UbuntuからVS Codeが立ち上げられないなど。

WSL2のUbuntuを20系にアップグレードしたい。

# どうやって

## 事前確認

{% raw %}
```
cat /etc/issue
```
{% endraw %}

## 準備

### インストール済みパッケージの最新化

{% raw %}
```
sudo apt update
sudo apt upgrade
```
{% endraw %}

### クリーンアップ

{% raw %}
```
sudo apt --purge autoremove
```
{% endraw %}

### アップグレードに必要なパッケージのインストール

{% raw %}
```
sudo apt install update-manager-core
```
{% endraw %}

## アップグレード

### 躓く原因になるパッケージの削除（環境依存？）

{% raw %}
```
sudo apt remove snapd
```
{% endraw %}

### アップグレード

{% raw %}
```
sudo do-release-upgrade
```
{% endraw %}

## 事後確認

{% raw %}
```
cat /etc/issue
```
{% endraw %}

# ちなみに

アップグレード時に -d オプションをつける例も見る。このオプションをつけると development release にアップグレードするようなので、そのつもりがないなら -d オプションはつけない方がよさそう。

{% raw %}
```
      -d, --devel-release
             If using the latest supported release, upgrade to the development release
```
{% endraw %}
