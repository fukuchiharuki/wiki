# キーワード
- Vim

# やりたいこと
バッファにあるファイルをショートカットでページ送りします。

# 方法
- _gvimrc

```
:map <silent> <C-p> :bp<CR>
:map <silent> <C-n> :bn<CR>
:map <silent> <F2> :bp<CR>
:map <silent> <F4> :bn<CR>
:map <silent> <F3> :call BufferList()<CR>
```

# 詳細

<dl>
<dt>ページ送り(次)</dt>
<dd>C-n, F4</dd>
<dt>ページ送り(前)</dt>
<dd>C-p, F2</dd>
<dt>バッファ一覧</dt>
<dd>F3</dd>
</dl>

F3 の分はプラグインが必要です。
