# キーワード
- Vim

# やりたいこと
バッファにあるファイルをショートカットでページ送りします。

# 方法
- _gvimrc
```
:map <silent> <C-p> :bp<CR>
:map <silent> <C-n> :bn<CR>
:map <silent> <F2> :bp<CR>
:map <silent> <F4> :bn<CR>
:map <silent> <F3> :call BufferList()<CR>
```

# 詳細
:ページ送り(次)|C-n, F4
:ページ送り(前)|C-p, F2
:バッファ一覧|F3

F3 の分はプラグインが必要です。
