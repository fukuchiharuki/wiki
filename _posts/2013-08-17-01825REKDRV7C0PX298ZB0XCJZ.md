# 概要
（同じブランチの）リビジョン間の差分を確認します（ファイル単位で）。

# 方法

```
$ svn diff -r (リビジョン前):(リビジョン後) svn://svnサーバ/リポジトリ/ブランチ --summarize
```

# 解説
summarizeがないとファイルの中も見ます。

(リビジョン後)はHEADと書くこともできます。
