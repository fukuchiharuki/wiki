# キーワード
- Mac
- Windows用
- キーボード

# したいこと

Windows用のキーボードをMacで使いたい。

# どうやって

[Karabiner-Elements](https://pqrs.org/osx/karabiner/)でキーの置き換えができる。

# ちなみに

僕はこうしてる。

| Windows用のキーボード → | Macでの役割 |
| --- | --- |
| CapsLock | Control |
| 無変換 | 英数 |
| 変換 | かな |
| 窓 | Option |
| Alt | 左Command |
| 右Ctrl | 右Command |
