# キーワード
- Linux
- スペック
- CPU
- メモリ
- ディスク容量

# 概要
Linuxマシンのスペック(CPU、メモリ、ディスク容量)を調べます。

# 方法
## CPU

{% raw %}
```
# cat /proc/cpuinfo
```
{% endraw %}

## メモリ

{% raw %}
```
# cat /proc/meminfo
```
{% endraw %}

## ディスク容量

{% raw %}
```
# df -h
```
{% endraw %}

# 解説
## CPU

*processor* と *physical id* と *cpu cores* を見てみます。

<dl>
<dt>processor</dt>
<dd>コアの通し番号、物理×1物理あたりのコア×HT(ハイパースレッディング)が分かる</dd>
<dt>physical id</dt>
<dd>物理の通し番号、物理が分かる</dd>
<dt>cpu cores</dt>
<dd>1物理あたりのコアが分かる</dd>
</dl>

## メモリ
*MemTotal* と *SwapTotal* を見てみます。

<dl>
<dt>MemTotal</dt>
<dd>メモリ量が分かる</dd>
<dt>SwapTotal</dt>
<dd>Swap量が分かる</dd>
</dl>

## ディスク容量
表になっているのでまんまです。
