# 目標

- Eclipseをインストールする
- Eclipseを起動する

# 手順
## アーカイブファイルをダウンロードする

1. 次のサイトの Downloads を選択する
  - http://www.eclipse.org/
1. 「Eclipse IDE for Java EE Developers」右の「Windows 32 Bit」を選択する
1. 下向き矢印を選択してダウンロードする
  - eclipse-jee-helios-SR2-win32.zip（2011-04-28時点）

## エクリプスをインストールする

1. ダウンロードしたアーカイブファイルを適当なディレクトリに解凍する

![Eclipseインストール00.JPG]({{ '/images/wiki/45636C697073652F57696E646F7773E381AB45636C69707365E38292E382A4E383B3E382B9E38388E383BCE383ABE38199E3828B_45636C69707365E382A4E383B3E382B9E38388E383BCE383AB30302E4A5047' | relative_url }})

### 初期化ファイルを編集する
eclipse.exeを実行すると次のエラーに遭遇することがあります。

- Failed to create The Java Virtual Machine

このとき、eclipse.ini に次の2行を追記します。※

{% raw %}
```
-vm
C:/Program Files/Java/jdk1.6.0_24/bin/javaw.exe
```
{% endraw %}

追記したら再度eclipse.exeを実行してください。

※次の行の上に追記してください。

{% raw %}
```
-vmargs
```
{% endraw %}
