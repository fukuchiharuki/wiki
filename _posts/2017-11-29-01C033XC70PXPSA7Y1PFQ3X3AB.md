# キーワード
- sed
- 拡張正規表現

# したいこと

sedで拡張正規表現を利用する。

# どうやって

「-E」オプションまたは「-r」オプションをつける。

```
$ sed -r 拡張正規表現
```

# ちなみに

「-e」オプションで利用できる標準正規表現では一文字以上を示す「+」が使えなかったり、グループ化するための「(」「)」にエスケープが必要だったりで、ちょっと不便。これまで標準正規表現も拡張正規表現もよく知らずに使っていた。どおりで、、という感じ。
