# キーワード
- Apache
- .htaccess
- .htpasswd

# 概要
.htpasswdを作成します。Apacheにはhtpasswdというコマンドが用意されているのでここではそれを利用します。

# 方法

{% raw %}
```
htpasswd -nb guest guest | grep -E '.' >> .htpasswd
```
{% endraw %}

# 解説
htpasswdのオプションはそれぞれ次の意味があります。

|  |  |
| --- | --- |
| n | 出力を標準出力にします。 |
| b | コマンドの引数でパスワードを設定します。 |

grepをかましているのは空行を作らないようにしているだけです。
