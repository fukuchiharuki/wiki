# キーワード
- Atom
- ショートカット
- ctrl-[
- エスケープ
- アウトデント

# 現象

「ctrl-[」で入力モードから抜けて、再度「ctrl-[」するとアウトデントしてしまう。連打する癖があるので結構辛い。

インデントの反対はアウトデントなの？

## （2016-11-08追記）

逆に、入力モード中にインデントしてしまう＠Ubuntu。

これについてはこちらの記事を参考にして解決できた。

- [欅樹雑記: Atomのvim-modeでCtrl+[が効かない&#65281;&#65288;keyboard-localizationプラグインと日本語キーマップのあれこれ&#65289;](http://blog.zelkova.cc/2016/02/atom-vim-mode.html)

# 原因

Atomにキーバインドが設定されているから。

# 対策

- File > Keymap...

で次を打ち込んでおく。

{% raw %}
```
'atom-workspace atom-text-editor:not([mini])':
  'ctrl-[': 'unset!'
```
{% endraw %}

# 備考

- File > Settings > Keybindings

で設定を見ることができる。
