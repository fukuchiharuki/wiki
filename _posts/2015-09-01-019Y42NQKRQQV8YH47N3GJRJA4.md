# キーワード
- FTPClient
- PASV
- パッシブモード

# 現象

FTPClientでFTP/putできない。環境は、ファイアウォールの都合上パッシブモードでしか送信できない。

エラーになるのは次のコード。

{% raw %}
```
client.storeFile(remote, local)
```
{% endraw %}

# 原因

次のコードでパッシブモードにしたつもりだができていない。

{% raw %}
```
client.pasv()
```
{% endraw %}

# 対策

こうすべきだったようです。

{% raw %}
```
client.enterLocalPassiveMode()
```
{% endraw %}
