# キーワード
- ネットワーク
- LAN

# 概要

ネットワーク(LAN)設定します。<br>
ここではプライベートで固定なIPアドレスを設定します。

# 方法

## ホスト名を設定する
ゲートウェイも。

- /etc/sysconfig/network
```
NETWORKING=yes
NETWORKING_IPV6=no
HOSTNAME={ホスト名}
GATEWAY={ゲートウェイのIPアドレス}
```

## IPアドレスを設定する

- /etc/sysconfig/network-scripts/ifcfg-eth0
```
DEVICE=eth0
BOOTPROTO=none
ONBOOT=yes
HWADDR={NICのMACアドレス}
IPADDR={設定するIPアドレス}
NETMASK=255.255.255.0
GATEWAY={ゲートウェイのIPアドレス}
TYPE=Ethernet
USERCTL=no
```

## ネットワークを再起動する

```
# /etc/init.d/network restart
```
