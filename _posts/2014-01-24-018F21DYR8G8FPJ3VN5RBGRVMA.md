# キーワード
- diff
- 標準出力

# やりたいこと
ふたつのコマンドが標準出力に出力する内容についてdiffかけます。

# 方法

```
$ {コマンド1} | ({コマンド2} | diff /dev/fd/3 -) 3<&0
```

# 詳細
1. コマンド1 の実行をファイルディスクリプタ 3 に出力し
1. diff がファイルディスクリプタ 3 を使用
1. コマンド2 の実行がパイプしているのでこれと比較

ということのようです。
