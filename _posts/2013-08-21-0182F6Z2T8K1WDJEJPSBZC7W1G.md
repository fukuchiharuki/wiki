# 現象
Tomcatを終了させようとして。

```
Error occurred during initialization of VM
Could not reserve enough space for object heap
Could not create the Java virtual machine.
```

Tomcatではなく、Javaがメモリ不足を言っている様子。

# 原因
メモリが足りてない。

# 対策
## とりあえず強制終了させる

```
$ ps -ef | grep java
```

Tomcatに関わるプロセスを見つけたら、殺す。

```
$ kill プロセスID
```

## ヒープの最大を指定してTomcatを動かす

Tomcatを起動させうユーザの環境変数CATALINA_OPTSに以下を設定する。

```
export CATALINA_OPTS='-server -Xmx64M'
```
