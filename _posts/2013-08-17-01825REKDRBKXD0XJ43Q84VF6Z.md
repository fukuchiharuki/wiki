# キーワード
- HTML/CSS
- after(疑似要素)

# 現象
Google ChromeとSafariでページ内リンクできない。

- HTML

{% raw %}
```
<div id="header"></div>

<a href="#header"><span></span></a>
```
{% endraw %}

- CSS

{% raw %}
```
a[href="#header"] > span:after {
  content: "このページのトップへ"
}
```
{% endraw %}

見栄えはアンカーだがリンクになってない。

# 原因
どうも疑似要素ではリンクできないくさい。

# 対策
JavaScriptで差し替えた。

{% raw %}
```
jQuery('a[href="#header"] > span').after("このページのトップへ").remove();
```
{% endraw %}
