# .bashrcと.bash_profileの作成

コマンド待ちの表示とエイリアスを設定した。これをしないとターミナルが使いづらい。

- $HOME/.bashrc

{% raw %}
```
PS1="[\u@\h \W]¥n$ "

alias ll="ls -l"
alias la="ls -la"
```
{% endraw %}

- $HOME/.bash_profile

{% raw %}
```
test -r ~/.bashrc && . ~/.bashrc
```
{% endraw %}
