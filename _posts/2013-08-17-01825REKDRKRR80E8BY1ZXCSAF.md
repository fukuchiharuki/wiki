# キーワード
- Linux
- du
- --max-depth

# 概要
どのディレクトリがたくさん使っているか、とか。

# 方法

```
# du -h --max-depth 1 /
```

# 解説
## -h
「7.0G」のような形式で使用量を表示する。

## --max-depth 1
深さを指定する。これがないとどこまでも再帰するので見づらい。

## /
最後に確認するディレクトリを指定する。この例はルートディレクトリ。
