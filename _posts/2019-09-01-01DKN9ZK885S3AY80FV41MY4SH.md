# apt
## aptのリポジトリを日本のものにする

```
$ cd /etc/apt
$ sudo sed -i.bak -e "s/http:\/\/archive\.ubuntu\.com/http:\/\/jp\.archive\.ubuntu\.com/g" sources.list
```

## 更新

```
$ sudo apt update
$ sudo apt upgrade
```

# 日本語化
## 言語設定

```
$ sudo apt install language-pack-ja
$ sudo update-locale LANG=ja_JP.UTF-8
```

## タイムゾーン

```
$ sudo dpkg-reconfigure tzdata
```

「アジア」>「東京」を選択する。
