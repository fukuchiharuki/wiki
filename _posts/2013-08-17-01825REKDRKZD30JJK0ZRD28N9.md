# ウィンドウ
- ウィンドウを上下に分割する

{% raw %}
```
C-x2
```
{% endraw %}

- ウィンドウを左右に分割する

{% raw %}
```
C-x3
```
{% endraw %}

- 分割したウィンドウをフォーカスしているウィンドウひとつに戻す

{% raw %}
```
C-x1
```
{% endraw %}

- 分割したウィンドウを移動する

{% raw %}
```
C-xo
```
{% endraw %}
