# キーワード
- sed

# やりたいこと
ファイルから行数を指定して行を抜き出します。

# 方法
```
$ cat {ファイル名} | sed -n '2,4p'
```

# 詳細
2 行目から 4 行目を p プリントする、ということのようです。
