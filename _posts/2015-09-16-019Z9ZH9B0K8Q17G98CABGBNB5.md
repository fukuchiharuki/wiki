# キーワード
- bash

# したいこと

ディレクトリ内のファイル一覧を取得してひとつずつ処理する。

# どうやって

{% raw %}
```
FILES=/path/to/files/*.txt
for FILE in $FILES
do
  echo $FILE
done
```
