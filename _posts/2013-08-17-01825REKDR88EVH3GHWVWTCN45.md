# 関連
- MySQL

# 現象
MySQLに対して"LOAD DATA LOCAL"でファイルインポートしようとすると次のエラーを得る。

{% raw %}
```
ERROR 1148 (42000) at line 1:The used command is not allowed with this MySQL version
```
{% endraw %}

# 原因
LOAD DATA LOCALが有効ではないから。

# 対策
MySQLログインにへのログインに次のオプションをつける。

{% raw %}
```
--local-infile=1
```
{% endraw %}
