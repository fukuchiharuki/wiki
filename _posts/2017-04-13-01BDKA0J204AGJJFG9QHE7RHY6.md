# 関連
- MySQL

# 概要
データベースを丸ごとダンプして別のデータベースにコピーする。

# 方法
データベースsrcからデータベースdstにコピーするものとして。

{% raw %}
```
$ mysqldump src -u**** -p**** --add-drop-table --extended-insert --no-autocommit > dump
$ mysql dst -u**** -p**** < dump
```
{% endraw %}

# 解説

<dl>
<dt>--add-drop-table</dt>
<dd>CREATE TABLE文の前にDROP TABLE文をつける</dd>
<dt>--extended-insert</dt>
<dd>複数行INSERT構文を使用する（→取り込むときに高速化）</dd>
<dt>--no-autocommit</dt>
<dd>テーブル毎のトランザクションにする（→取り込むときに高速化）</dd>
</dl>

# さらに

<dl>
<dt>--single-transaction</dt>
<dd>ダンプ中テーブルをロックしない</dd>
<dt>-t データベース名 テーブル名</dt>
<dd>データだけ</dd>
<dt>-d データベース名 テーブル名</dt>
<dd>スキーマだけ</dd>
</dl>
