# キーワード
- Angular2
- ngFor
- table
- tr

# したいこと

tableを置いていて、ひとつの要素が複数のtrで成るときngForをどう書くか。

```
<table>

  <tr>...</tr><!-- item1 -->
  <tr>...</tr><!-- item1 -->

  <tr>...</tr><!-- item2 -->
  <tr>...</tr><!-- item2 -->

</table>
```

# どうやって

templateを使います。

```
<table>

  <template ngFor let-item [ngForOf]="list">
    <tr>...</tr>
    <tr>...</tr>
  </template>

</table>
```

# ノート

```
*ngFor
```

とする書き方と結構違うから知らないと書けないよね。
