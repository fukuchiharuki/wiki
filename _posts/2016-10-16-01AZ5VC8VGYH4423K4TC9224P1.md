# グローバルにインストール

```
npm install -g <package>
```

グローバルにパッケージをインストールする。どのプロジェクトでも利用するときに。

# プロジェクトに閉じてインストール

```
npm install --save <package>
```

パッケージの情報をpackage.jsonに書いてインストールする。

# package.jsonに基づいてインストールする

```
npm install
```

package.jsonに従ってパッケージをインストールする。
