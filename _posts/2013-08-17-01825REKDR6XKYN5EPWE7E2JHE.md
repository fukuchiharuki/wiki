# キーワード
- ブラウザ
- キャッシュ
- POST

# 現象
ブラウザで戻ると「ドキュメントが有効期限切れです」(Firefox)になる。

# 原因
- POSTのレスポンスが
- キャッシュされないと

そうなる。

ブラウザの振舞いとしては妥当な結果。
POSTのレスポンスは一時的であると考えられるためそうなる。

# 対策
キャッシュさせるのを許すと回避できる。

ServletやJSPでなら、次の記述をしなければよい(消す)。
```
<% response.addHeader("pragma", "no-cache"); %>
```
