# キーワード

- Linux
- netstat
- ポート
- LISTEN

# 概要
LISTEN しているポートとプログラムを netstat を使って調べます。

# 方法

{% raw %}
```
# netstat -lanput
```
{% endraw %}

参考サイトによると「らんぷっと」と覚えるのだそうですｗ

# 解説
せっかくなのでオプションを調べました。man コマンドで。

<dl>
<dt>-l, --listening</dt>
<dd>接続待ち (listen) 状態にあるソケットのみを表示する。</dd>
<dt>-a, --all</dt>
<dd>接続待ち状態にあるソケットも、接続待ち状態にないソケットも表示する。</dd>
<dt>-n, --numeric</dt>
<dd>ホスト・ポート・ユーザーなどの名前を解決せずに、数字のアドレスで表示する。</dd>
<dt>-p, --program</dt>
<dd>各ソケットが属しているプログラムの PID と名前が表示される。</dd>
<dt>-u, --udp</dt>
<dd>UDPのポートを表示？</dd>
<dt>-t, --tcp</dt>
<dd>TCPのポートを表示？</dd>
</dl>
