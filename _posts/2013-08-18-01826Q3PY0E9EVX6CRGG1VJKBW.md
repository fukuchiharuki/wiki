# キーワード

- Linux
- Tomcat
- MySQL
- Subversion
- Redmine

# 概要
アプリケーションの起動方法と終了方法をよく忘れるのでメモ。

# 方法
## Tomcat

- 起動

{% raw %}
```
$ /usr/local/tomcat/bin/startup.sh
```
{% endraw %}

- 終了

{% raw %}
```
$ /usr/local/tomcat/bin/shutdown.sh
```
{% endraw %}

## MySQL

- 起動

{% raw %}
```
$ nohup /usr/local/mysql/bin/mysqld_safe &
```
{% endraw %}

- 終了

{% raw %}
```
$ /usr/local/mysql/bin/mysqladmin -uroot -p shutdown
```
{% endraw %}

## Subversion

- 起動

{% raw %}
```
$ svnserve -d -r /var/svn
```
{% endraw %}

- 終了

{% raw %}
```
$ kill (svnserveのプロセス番号)
```
{% endraw %}

## Redmine

- 起動

{% raw %}
```
$ nohup /usr/local/redmine/script/server -e production &
```
{% endraw %}

# 解説
パスはインストールの仕方によるので適当に読み替えること。
