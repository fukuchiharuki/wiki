# 概要
標準エラー出力を取得したいです。

# 方法
次のようにすると標準エラー出力を値として取得できます。

{% raw %}
```
tmp=`cp a b 2>&1`
```
{% endraw %}

# 解説
キモは次の箇所。

{% raw %}
```
2>&1
```
{% endraw %}

標準エラー出力(2)を標準出力(1)に抱き合わせているだけだとか。
分別して取得することはできないのかしら？
