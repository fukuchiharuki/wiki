# Configurations

「システム環境設定 > セキュリティとプライバシー > プライバシー > 入力監視」に次を追加。

- karabiner_grabber
- karabiner_observer

リストに挙がっていないときは端末で次を開いてドラッグ＆ドロップする。

{% raw %}
```
open /Library/Application\ Support/org.pqrs/Karabiner-Elements/bin
```
{% endraw %}

# Preferences

「Simple Modifications > 該当のデバイス」に次を追加（好みに応じて）。

- left_option -> left_command
- left_command -> left_option
- left_control -> caps_lock
- caps_lock -> left_control
- right_control -> right_command
- PCキーボードの無変換キー -> 英数キー
- PCキーボードの変換キー -> かなキー
