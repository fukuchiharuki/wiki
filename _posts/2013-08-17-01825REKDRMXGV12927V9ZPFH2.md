# キーワード
- Linux
- cron

# 概要
cron 設定します。

# 方法
実行させたいユーザーで次のコマンドを入力する。

```
$ crontab -e
```

vi が上がるので設定を編集する

```
# コメント
*/5 * * * * /your/sh/file/path
```

保存して終了する(vi の操作)

```
:wq
```

最後に設定を確認する

```
$ crontab -l
```

# 解説
専用のコマンドで編集を上げて確認します。
定時実行するユーザーは crontab -e を実行したユーザーです。
ここで、「*全部 root が実行ユーザーを添えて設定しろ*」という考え方もあるようです。

設定の書き方は参考サイトなどを参考にして。
