# キーワード
- cron
- mysql

# 現象

mysqlを叩くバッチ（シェルスクリプト）をcronでキックするようにしたが動いてないように見える。

# 原因

mysqlコマンドのパスが通っていないからだった。

## cronログを確認

{% raw %}
```
# cat /var/log/cron
```
{% endraw %}

キックしたことは確認できる。

## エラーを（ファイルに書き出すようにして）確認

{% raw %}
```
./bin/maintenance.sh: line 20: mysql: command not found
```
{% endraw %}

mysqlは関係なかった。

# 対策

フルパスでコマンドを書いてあげる。
