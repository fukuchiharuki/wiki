# キーワード

- JUnit
- 例外
- ExpectedException

# 現象
JUnitの例外テストでExpectedExceptionを使うとテストの初期化でエラーになる。

{% raw %}
```
initializationError
```
{% endraw %}

# 原因
ExpectedExceptionをpublicにしていなかった。

{% raw %}
```
@Rule ExpectedException thrown = ExpectedException.none();
```
{% endraw %}

# 対策
ExpectedExceptionをpublicにする。

{% raw %}
```
@Rule public ExpectedException thrown = ExpectedException.none();
```
{% endraw %}

# 備考
ここpublicなの？と思ったら、必要だったようです。
