# キーワード
- if
- test

# したいこと

いろんなif文を書く。

# どうやって

## 普通のif文

{% raw %}
```
if [ -e $LOCK ]
then
    exit 0
fi
```
{% endraw %}

## 単に1行に書く

{% raw %}
```
if [ -e $LOCK ]; then exit 0; fi
```
{% endraw %}

## 終了ステータスを利用する

{% raw %}
```
[ -e $LOCK ] && exit 0
```
{% endraw %}

## コマンドらしく書く

{% raw %}
```
test -e $LOCK && exit 0
```
{% endraw %}
