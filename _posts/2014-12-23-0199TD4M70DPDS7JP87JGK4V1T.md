# キーワード
- echo
- 色

# やりたいこと
[ OK ] や [ NG ] に色をつけて(echo)出力したい。

# 方法

```
#!/bin/bash

OK="\e[32mOK\e[m"
NG="\e[31mNG\e[m"
WARNING="\e[33mWARNING\e[m"

echo -e [ $OK ]
echo -e [ $NG ]
echo -e [ $WARNING ]

```
