# キーワード
- ディレクトリ
- 上書き
- cp

# やりたいこと
ディレクトリ a からディレクトリ b にその内容を上書きたい。ちなみに cp ではうまくできない。

- a

{% raw %}
```
a/
└ a.txt
```
{% endraw %}

- b(上書く前)

{% raw %}
```
b/
└ b.txt
```
{% endraw %}

- b(上書いた後)

{% raw %}
```
b/
├ a.txt
└ b.txt
```
{% endraw %}

# 方法

{% raw %}
```
$ ( cd a; tar cf - * ) | ( cd b; tar xf - )
```
{% endraw %}

# 詳細
tar でアーカイブして tar で展開する、データI/Oはパイプでやるということのようです。
