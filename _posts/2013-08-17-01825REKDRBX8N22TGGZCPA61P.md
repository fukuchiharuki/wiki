# キーワード
- JavaScript/jQuery
- animate
- scrollTop

# 現象
次のJavaScriptでページの先頭にスクロールしない。
```
jQuery('body:first').animate({
  scrollTop: 0
}, 100);
```

# 原因
bodyを指定してそうできるのはChromeだけだとか？

# 対策
こう書く。
```
jQuery('html:first, body:first').animate({
  scrollTop: 0
}, 100);
```
