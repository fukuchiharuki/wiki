# 関連

- sed

# 概要
入力と出力を同じにしてsedを使いたいとき。

# 方法

{% raw %}
```
sed -i -e (やりたいこと) (ファイル名)
```
{% endraw %}

# 解説
iオプションが「--in-place」ということなのだそうです。
