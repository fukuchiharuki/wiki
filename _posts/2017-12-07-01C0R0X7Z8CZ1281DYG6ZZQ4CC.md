# キーワード

- Windows
- bat
- フォルダ

# したいこと

途中のコマンド実行が成功しても失敗しても元のディレクトリに戻ってきたい。

なお、行頭で

{% raw %}
```
@cd /d %~dp0
```
{% endraw %}

を書いておくと、最初にバッチファイルのあるフォルダに移動しておけますね。bashでやるところの次です。

{% raw %}
```
cd $(dirname $0)
```
{% endraw %}

# どうやって

「&&」と「\|\|」の両構えにしておく。

{% raw %}
```
command &&^
cd /d %~dp0 ||^
cd /d %~dp0
```
{% endraw %}

# ちなみに
