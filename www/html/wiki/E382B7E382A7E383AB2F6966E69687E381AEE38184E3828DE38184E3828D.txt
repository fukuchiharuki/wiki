いろんなif文を書く。

* どうやって [#u269fded]

** 普通のif文 [#o41237b3]
 if [ -e $LOCK ]
 then
     exit 0
 fi

** 単に1行に書く [#e0ee915f]
 if [ -e $LOCK ]; then exit 0; fi

** 終了ステータスを利用する [#x3d2c727]
 [ -e $LOCK ] && exit 0

** コマンドらしく書く [#c9d35b67]
 test -e $LOCK && exit 0
